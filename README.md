# Monia lite
## How to deploy
### Deploy a node



1. clone the repo \
```git clone -b develop https://gitlab.partdp.ir/network/monia/new-monia/monia.git```

2. install npm packages \
```npm i```

3. copy `.env.example` as `.env.` \
```cp .env.example .env```

7. Run monia  \
```node index.js``` Or ```pm2 start index.js```
